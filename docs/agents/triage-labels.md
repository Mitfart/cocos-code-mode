# Triage labels

Use simple labels unless repo defines stricter GitHub labels later.

- `bug`: wrong behavior or crash
- `feature`: new editor/tool capability
- `docs`: README, prompt, or agent docs
- `build`: TypeScript/package/release pipeline
- `cocos`: Cocos Creator API/editor integration
- `utcp`: UTCP schema/provider/config behavior
- `needs-repro`: cannot act without reproduction
- `upstream`: likely belongs in or affects upstream repo
